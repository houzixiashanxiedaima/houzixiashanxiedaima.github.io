Markdown 是一种轻量级标记语言，非常适合写作。

## 基础语法

### 标题

使用 `#` 创建标题：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 强调

- **粗体**：`**文本**`
- *斜体*：`*文本*`
- ~~删除线~~：`~~文本~~`

### 列表

无序列表：

- 项目 1
- 项目 2
- 项目 3

有序列表：

1. 第一步
2. 第二步
3. 第三步

### 链接和图片

链接：`[文本](URL)`

图片：`![alt](图片URL)`

### 代码

行内代码：`` `code` ``

代码块：

\`\`\`javascript
function hello() {
  console.log('Hello, World!');
}
\`\`\`

### 引用

> 这是一段引用文字。
> 可以跨越多行。

### 表格

| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 数据1 | 数据2 | 数据3 |
| 数据4 | 数据5 | 数据6 |

### [VSCode 1.128.1 → 1.129.0](https://raw.githubusercontent.com/microsoft/vscode-docs/main/release-notes/v1_129.md) <sup>gpt-oss-120b</sup>

**新内容**
- 引入专用 Agent Host，代理会话在独立进程运行，可在多个窗口共享
- 新增 Agents 窗口的实验性编辑面板，将编辑器与详情区合并为单一停靠窗格
- 为 Agent Host 会话提供会话管理工具，可列举、创建、读取和发送消息
- 支持在聊天前缀 “!” 中执行终端命令，编辑器和 Agents 窗口均可使用
- 支持 BYOK（自带密钥）模型的 Copilot 代理，可在 Agents 窗口使用
- 实验性将 Prompt 文件迁移为 Skills，提供迁移界面和兼容性建议
- 编辑器工具栏新增 “Reopen Editor With” 可直接切换编辑器打开文件或 diff
- 预览现代化 UI（Experimental），可通过设置开启
- 为 GitHub Enterprise 用户提供 Copilot 认证支持，支持在企业实例登录
- 自定义编辑器默认不再用于 diff 与合并编辑器，需手动打开
- Agents 窗口改进：记住新会话默认模式和审批选项
- Agents 窗口改进：新增 “New Worktree” 复选框以使用 Git worktree 隔离

**Breaking Changes**
- 自定义编辑器默认不再处理 diff/merge，可能导致行为变化，需要手动切换
- Prompt 文件迁移为 Skills，旧 Prompt 文件不再支持，需迁移

**问题修复**
- 修复聊天语音在问题轮播中的答案展示问题
- 修复 Modern UI 编辑器标签栏在完整标签时的颜色显示错误
- 修复 Chat 模型设置相关的错误
- 若干其他零星错误修复

Zulu JDK 17 17.0.19 → 17.0.20 gpt-oss-120b

新内容

  • 移除 Zulu 8 中的 Lucida Monotype 字体
  • 引入 TLS 1.3 后量子混合密钥交换（JEP 527）
  • 加强证书解析（JDK-8381796）
  • 新增自动检查点功能（CRaC）
  • 更新 IANA 时区数据版本

安全修复

  • 包含关键补丁更新（CPU），修复安全漏洞

Breaking Changes

  • 移除 Lucida Monotype 字体，可能导致依赖该字体的应用需迁移

问题修复

  • PSU 包含的非安全性 bug 修复
  • 删除已弃用的选项

Zulu JDK 21 21.0.11 → 21.0.12 gpt-oss-120b

新内容

  • 新增对 TLS 1.3 的后量子混合密钥交换支持（JEP 527）
  • 改进证书解析功能
  • 引入自动检查点（Coordinated Restore at Checkpoint）
  • 更新 IANA 时区数据版本
  • 提供不含和含 OpenJFX 的 Azul Zulu OpenJDK 构建

安全修复

  • 本次 CPU 更新包含安全漏洞修复

Breaking Changes

  • 移除 Zulu 8 中的 Lucida Monotype 字体，可能导致依赖此字体的应用需更换
  • 删除已废弃的选项，需相应调整配置

问题修复

  • 包含关键错误修复
  • PSU 版本合并了 CPU 的安全修复并加入其他非安全 bug 修复

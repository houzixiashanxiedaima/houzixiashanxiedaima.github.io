### [VSCode 1.129.0 → 1.129.1](https://raw.githubusercontent.com/microsoft/vscode-docs/main/release-notes/v1_129.md) <sup>gpt-oss-120b</sup>

**新内容**
- 引入专用 Agent Host，支持在独立进程中运行 Copilot、Claude、Codex 等代理
- 在 Agents 窗口新增实验性编辑面板，将编辑器与详情区合并为可停靠的单一面板
- 为 Agent Host 会话提供会话管理工具，可列举、创建、读取和发送消息到其他会话
- 支持在聊天中使用 ! 前缀将内容作为终端命令执行
- 在 Agents 窗口的 Copilot 代理中支持 BYOK（自带密钥）模型
- 将 Prompt 文件迁移为 Skills（实验性），提供迁移界面
- 编辑器工具栏新增“重新打开编辑器”功能，可直接切换编辑器
- 预览现代化 UI（实验性），可通过设置启用
- 为 GitHub Enterprise 用户提供在 Agent Host 中使用 Copilot 的认证支持
- 自定义编辑器默认不再用于 diff 与 merge 编辑器，保持使用内置编辑器
- Agents 窗口新会话流改进：记住上次选择并提供工作树复选框

**Breaking Changes**
- 自定义编辑器默认不再用于 diff 与 merge 编辑器，可能导致行为变化
- 实验性编辑面板需手动启用，可能影响现有布局

**问题修复**

### [VSCode 1.128.0 → 1.128.1](https://raw.githubusercontent.com/microsoft/vscode-docs/main/release-notes/v1_128.md) <sup>gpt-oss-120b</sup>

**新内容**
- 新增多聊天会话支持 Claude 代理，支持在同一会话中比较、分支和并行运行
- 在 Agents 窗口可在未选择工作区的情况下启动聊天
- 预览模式下子代理聊天为只读，便于查看进度
- 为 Agents 窗口的聊天提供键盘快捷键，实现快速切换和管理
- Copilot Vision 正式发布，支持在聊天中直接粘贴、拖拽图片和 PDF
- 支持 BYOK（自带密钥）模型在代理主机会话中使用（实验性）
- 可为自定义端点模型配置 temperature 与 top_p 参数
- 新增 modelOptions 对象，可覆盖或取消模型默认参数
- 可配置 BYOK 的默认实用模型
- 支持深度链接直接打开指定聊天及其工作区
- 可配置集成浏览器标签的打开位置
- 引入 OS 级键盘快捷键，即使 VS Code 未聚焦亦可生效
- 支持通过 OpenTelemetry 管理 Copilot 遥测导出

**问题修复**

jdx/mise v2026.7.0 gpt-oss-120b

新内容

  • 默认启用 env 的 shell 展开，可通过 env_shell_expand=false 关闭
  • 新增 monorepo 支持：mise install/ls —monorepo 可统一管理跨根目
  • 任务系统支持根级 mount 语法，简写 mount “cmd” 可提升生成任务命令
  • 为单个工具提供 github_attestations 选项，可关闭 GitHub Artifact
  • brew 配方修剪改为基于前缀清单，可删除未在 bootstrap.packages 中的链接公式

问题修复

  • 安装时在 —locked 模式下遵循 lockfile 记录的后端
  • 安装器提供更清晰提示，建议在不支持的架构上使用 cargo install —locked mis
  • install-into 在目标目录非空且未加 —yes 时拒绝覆盖
  • brew-cask 处理原始可执行二进制，解析 $APPDIR 路径，使用 ditto 复制避免 m
  • http 处理避免 netrc 凭证覆盖显式 forge token，修复私有 GitHub 发布资

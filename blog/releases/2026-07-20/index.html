alyssaxuu/screenity v4.6.0 - v4.6.1 gpt-oss-120b

新内容

  • 捕获分辨率提升至编码尺寸的 2 倍并下采样，细节更丰富
  • 默认使用 MP4/H.264 代替 WebM/VP9 进行录制
  • 在阻止摄像头 / 麦克风的站点（如 facebook.com）显示权限警告
  • 当站点阻止自定义区域录制时，预先禁用该功能
  • 系统音频警告 Toast 移至底部，避免被 Chrome 屏幕共享选择器遮挡
  • 硬件编码器不可用或忙碌时，自动切换至软件编码器重试
  • 计算机从睡眠唤醒后，录制恢复速度更快
  • 裁剪后录制保持原始格式，不再错误转换为 WebM 并错误标记

问题修复

  • 修复部分 Windows 机器因硬件编码器问题导致录制无法启动的错误
  • 修复在较慢机器上因监控器误判导致录制在启动后几秒自动停止的问题
  • 修复在实际正常的录制上错误显示 “无法加载你的录制” 提示的情况，包括取消屏幕选择后
  • 修复区域录制回退至 VP8 丢失前几秒的问题；修复超宽显示上比特率维度计算错误

hugohe3/ppt-master v4.0.0 gpt-oss-120b

新内容

  • 新增真实幻灯片母版与布局支持，导出时继承主题颜色、字体并生成活页码字段
  • 支持完整页面背景（纯色 / 渐变）导出为 PowerPoint 原生背景
  • 引入可复用的图表 SVG 模板库并保持渲染保真度，实现令牌使用效率提升
  • 每页执行上下文复用未变更引用，压缩执行锁合约，提升令牌利用率
  • 新增提示预算与治理 lint（prompt‑audit），导出后收据安全令牌
  • 新增 Create Template 路由，将 PPTX、图片、PDF、文档或品牌资产转为可复用模板（Brand、Layout、Deck）
  • 新增 Fill Native PPTX 路由，直接在原始 PPTX 模板上打补丁，无需重新生成 SVG
  • 支持模板使用模式（自适应或严格）并明确确认
  • PPTX 导入时读取 SmartArt 语义结构，改进表格 / 图表重建及 XY 轴语义
  • 转换器支持 CSS 颜色透明度、组透明度以及图片透明度的原生保留
  • 启用所有原生线端箭头形状，统一动画与转场行为，旁白导出添加 _narrated 后缀
  • 规划与确认 UI 提供 AI 生成的设计候选与 18 种风格双目录，支持规格版本化并强制首页检查
  • 图像生成新增参考图像单图编辑模式（OpenAI 后端），用于管道外的重新样式 / 迭代
  • 社区改进：对非 ASCII 页面名称进行 URL 编码，提升实时预览兼容性

问题修复

  • 修复未排序的渐变停靠点、过大瓦片图像填充以及损坏源构造的本地恢复问题，提升导入鲁棒性
  • 修复非 ASCII 页面名称在实时预览中的 URL 编码问题

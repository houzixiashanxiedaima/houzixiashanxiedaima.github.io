jdx/mise v2026.6.12 GPT-OSS 120B

新内容

  • 新增 bootstrap 子命令 —skip，可按阶段、逗号分隔选择执行
  • 新增 http 支持通过 checksum_url、checksum_expr、checksum_a
  • 新增 aqua 能解析并验证 checksum.replacements、checksum.mini
  • 新增 registry 条目：aspire、ldc、nub

问题修复

  • 修复升级时在 minimum_release_age 生效时未正确移除旧版本的问题
  • 修复部分安装失败后未重新构建运行时符号链接和 shim 的问题
  • 修复在无会话 D-Bus 环境下未检测到 systemd 用户管理器的问题
  • 修复任务配置冲突时未优先使用高优先级 TOML 的问题
  • 修复任务执行时空 shell 导致崩溃，改为返回错误并非零退出码

steipete/CodexBar v0.37.1 - v0.37.2 GPT-OSS 120B

新内容

  • 诊断：将已编辑的提供者报告写入文件，包含平台和应用版本信息
  • CLI 服务器：在 /health 接口报告启动构建版本，以便客户端检测更新后过时的辅助进程

问题修复

  • Claude：在速率限制后短暂暂停后台 CLI 使用探测，手动刷新仍可用
  • Codex OAuth：发布已刷新且已设置私有文件权限的 auth.json 凭证
  • 提供者端点：在附加凭证前拒绝不安全的 Deepgram、z.ai、Xiaomi MiMo 覆盖
  • Azure OpenAI：在附加 API 密钥前拒绝不安全的端点覆盖，并显示可操作错误
  • 修复 MiniMax 在编码计划页面仅显示粗略使用时的详细 token 计划窗口恢复
  • 修复成本历史子菜单行的多余提示
  • 修复刷新菜单时保持打开并显示进度
  • 修复内存压力导致的 actor 隔离崩溃
  • 修复 Antigravity 在 OAuth 识别账户但配额接口拒绝时显示限制不可用

Homebrew/brew 6.0.3 GPT-OSS 120B

新内容

  • 允许在构建沙箱中信任 tap
  • livecheck 新增 compressed 选项
  • 在安装计划中标注可升级依赖
  • 支持在 Brewfile 中使用 tap trust
  • 读取用户 Git 配置以支持私有下载

问题修复

  • 跳过无效的 Cask 条目，避免错误
  • 在安装时跳过自冲突，提升成功率
  • 即使 skip_link 仍创建 opt 链接
  • services 检查 —sudo-service-user 参数存在性
  • 处理源修改时间的 Git 全局配置为空的情况

  • 优化大段纯文本翻译处理
  • 优化繁体中文文案、同步设置标签和接口测试目标语言逻辑

问题修复

  • 修复划词翻译弹窗相关问题，提升稳定性
  • 修复纯文本内容翻译卡顿或分段异常
  • 修复用户脚本设置页启动与 GM 桥接兼容性，支持原生和旧版 GM API
  • 修复渐变样式背景色、设置页 URL 匹配和开发环境选项页地址问题

ffuqiangg/build_openwrt 2026.06.18 GPT-OSS 120B

新内容

  • 新增 Bandix 流量监控插件
  • 使用 podman 替代 docker
  • OpenWrt 与 ImmortalWrt 固件升级至 25.12（6.12 内核）并启用 apk
  • iStoreOS 固件切换至 istoreos-24.10 分支并使用 fw4
  • iStoreOS 与 LEDE 固件升级至 6.6 内核，hy2 节点支持并自动处理 quic-go

Breaking Changes

  • 移除 DDNS、daed、带宽监控、FTP 服务器插件
  • podman 替代 docker，可能导致原有 Docker 脚本失效
  • sing-box 核心锁定在 1.12.x，升级至 1.13.x 以上会导致 homeproxy 插

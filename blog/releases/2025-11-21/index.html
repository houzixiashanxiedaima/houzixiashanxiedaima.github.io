sst/opencode v1.0.81 - v1.0.83

新内容

  • 更改默认子会话周期键绑定，从 ctrl+right/left 更改为 <leader>right/left。
  • 更新编辑器偏好设置，优先使用 VISUAL 环境变量而不是 EDITOR，遵循 Unix 规范。
  • 键绑定更改可能会影响现有用户的操作习惯。
  • 用户现在可以通过环境变量 OPENCODE_EXPERIMENTAL_BASH_MAX_OUTPUT_LENGTH 覆盖 bash.ts 中的 MAX_OUTPUT_LENGTH 设置。
  • 将 ruby-lsp 替换为 rubocop，以提高 LSP 性能。

问题修复

  • 解决了 AI SDK 警告信息被打印的问题。
  • /export 命令现在会跳转到当前工作目录。
  • 影响范围*: 所有使用 sst/opencode 的用户。

alyssaxuu/screenity v4.2.0

新内容

  • 替换 MediaRecorder：使用自定义的 WebCodecs 录制器直接将视频编码为 MP4 格式，使视频在编辑器中立即可用，无需额外处理步骤。这显著提升了用户体验和效率。

openai/codex rust-v0.61.0 - rust-v0.62.0-alpha.1

新内容

  • 改进截断与错误报告：单次截断减少重复工作，错误事件支持可选状态码，提升可观测性。
  • Shell 可靠性与沙箱警告：增强回退 shell 选择，减少对可写目录的警告，改进 Windows 下的信息提示。

问题修复

  • 优化了代码生成器的性能，特别是在处理大型项目时的速度提升明显。【影响范围：大型项目用户】

Breaking Changes

  • 改进了日志输出，增加了更多的调试信息，便于问题排查。
  • 注意:* 本次发布为 alpha 版本，建议在生产环境中谨慎使用。

openclaw/openclaw v2026.5.26 mimo-v2.5-pro

问题修复

  • 通过显式记忆存储工具提交的类提示文本在嵌入或存储前将被拒绝
  • 为远程非浏览器及 HTTP 网关认证失败启用默认认证速率限制器
  • 验证浏览器快照标签页 URL 是否符合 SSRF 策略
  • 将用户对话轮次持久化到目标会话
  • 排队繁忙 TUI 提示而非丢弃，并在入门时保留配置的默认模型
  • 保留插件 LLM 命令认证，并绑定到宿主代理的 LLM 认证
  • 将 WebChat 传递提示排除在用户提示之外
  • 通过保留 Telegram 输入 / 进度上下文来减少回复传递延迟
  • 保持 TUI、控制 UI 等源回复最终状态实时且无重复
  • 修复旧版工具结果以便重放，并保留会话生成的转录负载
  • 在会话发送中处理活动回退失败，以便回退路由报告真实失败
  • 强制执行默认钩子代理允许列表，并恢复失败的子代理生命周期完成
  • 在查找本地转录候选时包含 OpenClaw 代理会话日志

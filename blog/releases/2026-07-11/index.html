linkwarden/linkwarden v2.15.0 - v2.15.1 glm-4.7-flash

新内容

  • 大幅升级移动端并提供更轻量的 Web 应用
  • 支持在阅读视图中高亮文本，可选四种颜色并添加备注
  • 新增 “笔记与高亮” 视图，可快速浏览并跳转到标记位置
  • 引入离线模式，可在设置中开启 “离线保存”，后台下载所有保存格式
  • 长按链接可打开链接详情页，集中展示链接信息
  • 可自定义阅读视图的字体、字号、行高和背景颜色
  • 内存占用显著降低，空闲内存约减半至约 350 MB
  • Docker 镜像体积减半，约从 3 GB 降至约 1.5 GB
  • 支持任意 OpenID Connect 身份提供商的通用 OIDC 集成

安全修复

  • 本次发布包含大量安全加固，强烈建议升级至 2.x 版本

问题修复

  • 修复若干小问题

steipete/CodexBar v0.42.0 gemma-4-31b

新内容

  • 新增 Agent Sessions，支持在菜单和 CLI 中发现并管理本地或 SSH 连接的 Codex 和 Claude Code 会话
  • 新增 Wayfinder，支持本地网关健康状况、路由、节省额度及延迟用量监控
  • 菜单栏新增选项，可在配额耗尽时显示重置倒计时
  • 新增配额预警功能，支持对 Codex 和 Claude 会话及周限制进行预测性步调提醒
  • Codex 支持 GPT-5.6 Sol、Terra 和 Luna 定价，包括长上下文、优先级及缓存自动重定价
  • Codex 新增 Spark 配额行显示及隐藏选项
  • Claude CLI 现在可显示模型范围的周限制，且不与 Web 限制重复
  • Kimi K2 在旧版额度页面新增用量仪表盘快捷方式
  • 完善 Azure OpenAI、Perplexity、Mistral 和 Qoder 的设置与故障排除文档
  • 重构设置界面，优化布局、标准化标签并采用新的侧边栏设计

问题修复

  • 优化刷新机制，确保手动刷新响应流畅且后台任务有序执行
  • 修复菜单显示问题，防止已完成的卡片在无关任务运行时卡在刷新状态
  • 修复 macOS 27 上显示设置的交互问题
  • 修复 Codex 账户隔离问题，防止不同账户间缓存配额或身份响应导致错误重置提醒
  • 修复 Claude CLI 的重置时间戳解析，确保正确处理夏令时和闰日
  • 修复 CLI 服务器在超时时的请求堆叠问题
  • 修复 Widget 在 Token 成本行滞后时的显示与重试机制
  • 修复 Ollama 的 AuthKit 会话识别及 API 密钥验证
  • 修复 Kimi K2 API 密钥缺失或空白时的错误报告
  • 修复 MiMo 本地回退缓存过期标记显示
  • 修复 Gemini 付费层级标签显示及 Ollama 登录重定向问题
  • 修复成本历史记录的菜单稳定性及滚动显示
  • 修复加泰罗尼亚语的翻译完整性与一致性

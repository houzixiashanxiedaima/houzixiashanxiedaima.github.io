  • 使代理路由更安全，提升 CLI 运行可靠性
  • 移除对所有模型强制启用 multi-agent v2 的限制
  • 自动从模型目录选择合适的 multi-agent 模式，避免不支持的行为
  • 完整保留 Hephaestus 指令，不再在 SessionStart 时截断 md 内容
  • OpenCode CLI 现在能够解析并使用配置的代理显示名称进行调度
  • 放宽安装集成超时时间
  • 加速 git‑master 异步解析
  • 注入静态规则体代替引用
  • 持久化自动更新检查状态
  • 加强发布安全检查

问题修复

  • 停止在配置中强制启用 multi_agent_v2
  • 修复 SessionStart 注入时截断 hephaestus.md 的问题
  • 修正 CLI 运行时尊重配置的代理名称
  • 修复 CLI 调度时正确分发已注册的代理名称
  • 修复 GPT ultrawork 横幅和意图声明仅在首轮生效的问题
  • 修复团队创建参数为 null 时的验证错误
  • 排除子代理导致的 JSON 错误误报
  • 拒绝规则引擎中使用绝对距离路径
  • 处理 null 值的 team_create 选项视为缺省

Breaking Changes

  • 重命名跨包检测、插件 / 配置兼容性及安装入口的迁移更新
  • 调整任务系统默认行为，使省略配置在不同运行时路径下保持一致

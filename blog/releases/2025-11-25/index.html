sst/opencode v1.0.107 - v1.0.109

问题修复

  • MCPs 按字母顺序显示：侧边栏中的 MCPs 现在按字母顺序排列，便于查找和管理。
  • 持久化明暗模式：用户的明暗模式选择将被保存，下次打开时自动应用。
  • 文件侧边栏优化：改进了文件侧边栏的显示，提升了用户体验。
  • 增强模型处理：增加了显式的模型回退处理机制，防止在模型不可用时直接调用 opencode 提供商。
  • 优化标题生成：改进了对话标题的生成逻辑，提供了更明确的指南以创建有意义的对话标题。

lwouis/alt-tab-macos v7.35.0

问题修复

  • [优化] 通过减少不必要的后台进程，降低了 CPU 和内存占用，提高了应用的能效比。
  • * 影响范围:** 所有使用 macOS Sonoma 及以上版本的用户将显著受益于此次更新。

RooCodeInc/Roo-Code v3.34.2

问题修复

  • 支持 Gemini 3 模型的 reasoning_details 格式

zed-industries/zed v0.214.3-pre - v0.213.7

新内容

  • 工作区面板的即时加载优化适用于所有用户，特别是那些需要频繁切换面板的用户。

问题修复

  • 通过优化启动流程，显著提升了 Windows 用户的启动速度。(#43249)
  • * 影响范围 **: 所有 Windows 用户
  • 此版本主要集中在提升 Windows 平台的用户体验，确保启动更加迅速和流畅。
  • 此版本主要解决了预提交钩子长时间运行导致的提交问题，确保用户能够顺利使用 Zed 进行代码提交。影响范围: 所有使用 Zed 进行代码管理的用户。

Breaking Changes

  • 工作区面板即时加载 (#43426): 工作区中的面板现在会在准备好后立即显示，提高了用户体验和工作效率。
  • 所有使用 Opus 音频编解码器的用户将受益于新的版本支持。
  • 工作区用户将体验到更流畅的面板加载过程。
  • 工作区面板即时加载：改进了工作区面板的加载机制，确保面板在准备好后立即显示，提高了用户体验。#43422

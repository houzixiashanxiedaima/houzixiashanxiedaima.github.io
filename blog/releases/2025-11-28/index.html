zed-industries/zed v0.214.6

问题修复

  • 当 ACP 代理初始化超时时，将显示错误消息
  • 提升了软件稳定性，用户在特定操作下不再遇到程序崩溃

新内容

  • 增加 3 个应用，包括虫虫助手、QQ 音乐车机版和 UU 远程
  • 更新提示和权限提示规则各增加 1 条

RooCodeInc/Roo-Code v3.34.7 - v3.34.8

新内容

  • 支持 Anthropic 提供商的原生工具调用
  • 支持 z.ai 和 Moonshot 模型的原生工具调用
  • 添加提供商欢迎后的云引导屏幕

问题修复

  • 仅影响使用 new_task 工具的用户
  • 改进 OpenRouter 工具调用处理
  • 提前创建父目录以防止写文件时的 ENOENT 错误

Breaking Changes

  • 禁用 claude-code 提供商的原生工具和温度支持

jdx/mise v2025.11.9 - v2025.11.10

新内容

  • 使用 cargo 后端支持 tokei 的最新版本
  • 添加 SonarSource/sonar-scanner-cli 工具
  • 添加 charmbracelet/crush 到注册表
  • aqua 添加 symlink_bins 选项以过滤暴露的二进制文件
  • (hook-env) 添加快速路径以跳过未更改时的初始化

问题修复

  • aqua 在模板解析器中跳过管道符号前的空白字符

continuedev/continue @continuedev/openai-adapters@1.32.0 - v1.2.13-vscode

新内容

  • 添加对 Gemini 3 Pro 预览的支持
  • 添加对 Gemini 思维签名的支持
  • 升级 Opus 4-5
  • 添加对 Grok 4.1 快速的支持
  • 添加对 GPT 5.1 的支持
  • 添加 JCEF 重载操作
  • 增加 Klavis AI 的 Strata 至 MCP Cookbook
  • 增加 OpenAI 适配器 API 支持文档
  • 调整默认 CLI 配置文件路径

问题修复

  • 解决历史记录索引超出最大长度导致替换失败的问题
  • 提高图像 URL 解析的健壮性
  • 解决 MCP 密钥加载问题
  • 日常使用者需注意默认配置文件路径的变化，其他影响较小

Breaking Changes

  • 测试版本，7 天后若无重大问题将升级为稳定版

anghunk/linuxdo-scripts 1.5.11

新内容

  • 使用 html2canvas 替换 snapdom 实现图片捕获与下载

xchacha20-poly1305/husi plugin-mieru-v3.24.0-0

新内容

  • 支持新的加密算法，提升数据传输安全
  • 增加日志详细程度选项，便于调试
  • 优化连接管理，减少延迟

问题修复

  • 解决高负载下内存泄漏问题
  • 日常使用者需要关注新加密算法的启用

sst/opencode v1.0.118 - v1.0.119

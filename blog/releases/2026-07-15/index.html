jdx/mise v2026.7.6 gpt-oss-120b

新内容

  • 新增 task 支持每任务输出样式字段，可独立于详细程度组合前缀和静默等样式
  • 新增 sandbox 全局 deny 默认设置（deny_all、deny_read、deny_write、deny_net、deny_env）
  • 新增 bootstrap 对 Linux Flatpak 包的支持
  • 新增 oci 支持通过 mise oci build —copy 将主机路径复制为可复现的镜像层
  • 新增 doctor 在 PATH 中更早的可执行文件遮蔽 mise shim 时给出警告
  • 新增 aqua 支持通过认证 GitHub API 获取私有 github_content 与 github_archive 包
  • 新增 task 支持任务级超时时间使用 Tera 模板渲染
  • 新增 http_download_timeout 设置，限制下载总耗时（默认 30 分钟）
  • 新增 registry 条目 git-town 与 bundler
  • 性能提升：aqua 对标记为私有的包跳过公共可达性探测
  • 文档更新：aqua 添加本地自定义注册表说明
  • 文档更新：config 澄清子版本算术规则

Breaking Changes

  • task 的 —quiet /quiet = true / MISE_QUIET=1 不再折叠输出为无前缀交错，而是保留样式…
  • task 的 usage_* 变量改为调用本地作用域；原先隐式注入的工作流需通过 env= 声明输入

问题修复

  • 修复多项问题，包括 backend、config、task、s3、bootstrap、erlang、github、npm、cargo、registry…

  • 引入颜色主题、服务器标签页等 UI 改进
  • 在 v2 响应中公开会话位置数据（SDK）

问题修复

  • 修复编辑操作拒绝松散匹配，防止误覆盖代码或误删文件
  • 修复 Bedrock 会话在模型响应前挂起的问题
  • 修复外部编辑器回退，在附加模式下项目路径不可用时仍可打开会话
  • 修复标题栏标签溢出导致的剪裁和滚动拖动异常
  • 修复 shell 取消竞争导致的异常
  • 修复 OpenAI WebSocket 会话卡死问题
  • 修复 Windows 路径规范化在迁移存储中的错误
  • 修复粘贴靠近宽字符时提示被破坏的问题
  • 修复 GitHub 在缺少作者身份时拒绝提交的错误

firebase/firebase-tools v15.22.1 GPT-OSS 120B

ellite/Wallos v4.9.6 GPT-OSS 120B

问题修复

  • 加强 OIDC 状态验证和会话轮换

code-yeongyu/oh-my-openagent v4.13.0 GPT-OSS 120B

新内容

  • 将 Ultimate Browsing 纳入共享技能包，实现分层搜索
  • 引入分层 Insane Search：API 探测、归档回退、Jina、RSS 等路径
  • Ultraresearch 增加 claim‑ledger 验证门用于非代码研究
  • 支持研究去重并通过 Ultimate Browsing 升级受阻来源
  • 更新任务与工具行为，包括 delegate‑task 合约和运行时注册

问题修复

  • 修复 LazyCodex 发布负载路径错误
  • 将 LazyCodex 包大小控制在 npm 限制以下
  • 加固父进程唤醒恢复和提供者耗尽回退
  • 改进 Windows 测试可靠性和清理安全性
  • 修复安装 / 发布工作流的序列化问题

Breaking Changes

  • 重命名跨包检测、插件 / 配置兼容性和安装界面，需迁移
  • 兼容性行为变更可能影响现有插件

MHSanaei/3x-ui v3.4.0 GPT-OSS 120B

新内容

  • 每节点独立出站路由并支持 mTLS、哈希 + zstd 传输及节点指标
  • 新增通知事件总线，支持 Telegram 与 SMTP 订阅及内存阈值警报
  • 引入托管主机功能，可为每个主机单独覆盖订阅链接
  • 订阅引擎升级，支持动态备注变量、完整 XHTTP 映射及隐藏服务器设置
  • 扩容至 5‑10 万客户端，提升流量处理、批量操作及数据库索引性能

问题修复

  • 自动清除旧版 panelProxy 与 tgBotProxy 配置
  • 通过 panic‑recover 机制提升定时任务稳定性

Breaking Changes

  • IP 限制改为依赖 fail2ban，自动安装并不再解析 access.log
  • 升级后旧的 panelProxy /tgBotProxy 设置将被清除

问题修复

  • 修复 Go Agent 计数器重置导致流量统计不足

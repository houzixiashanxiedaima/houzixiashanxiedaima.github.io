jdx/mise v2026.7.3 - v2026.7.4 gpt-oss-120b

新内容

  • mise bootstrap 与 dotfiles 正式稳定，脱离实验模式
  • 支持在 mise.run 安装脚本中跳过已存在的重新安装（MISE_INSTALL_SKIP_IF_EXISTS）
  • 安装路径检查仅针对已解析的路径，默认行为保持不变
  • 安装过程并行检测缺失版本，加速大型工具集的检测
  • 注册表 Scala 简写解析更新为官方 fork
  • 新增 Aqua Registry 包 sderosiaux/launchdeck、syntax-sh/lexicon-releases
  • 恢复 Linux arm64 glibc 兼容性，确保 arm64 tarball 可用
  • 新增插件声明系统依赖并通过 system_deps 设置控制行为，缺失依赖会在 doctor 与 bootstrap 中显示
  • 为 Homebrew Cask 引入生命周期钩子，支持预检 / 后检脚本并改进安装可靠性
  • 新增终端宽度覆盖选项，可通过 MISE_TERM_WIDTH 或 COLUMNS 强制设置输出宽度
  • 将大量工具（scala、groovy、mongodb、emsdk、teleport 等）迁移至 vfox 后端，提升注册表一致性
  • 改进源码编译工具的构建智能化，实现更高效的构建流程

问题修复

  • Rust 后端在跳过安装前会查询 rustup，确保缺失组件 / 目标被重新安装
  • task_source_files () 在使用使用参数和重新渲染脚本时恢复可用
  • mise upgrade —minimum-release-age 不再误报 “新版本被忽略”
  • 构建时将 aarch64-unknown-linux-gnu 交叉镜像固定标签并加入 glibc 检查
  • 若干文档修复（bootstrap 页面、CLI 参考链接、样式一致性）
  • 修复升级时 mise upgrade —bump 重写锁文件条目导致平台元数据丢失的问题，现会重新锁定缺失平台信息的条目
  • 修复 Homebrew Cask 中二进制通过 pkg 安装后路径绝对路径的情况，现可正确安装并通过 caskroom 链接
  • 修复传统 vfox 插件下载位置，现存入 mise 的每工具下载目录并在安装后清理
  • 修复 Homebrew Cask Shim 未随 crate 发布导致的 cargo 验证失败
  • 文档更新：优化登录页文案和视觉

zed-industries/zed v1.10.0 gemma-4-31b

新内容

  • 新增 llama.cpp 作为语言模型提供商

  • 优化 Git 性能，防止多个远程操作同时启动，并改进了 diff 视图与行号颜色
  • Go 语言新增内置类型和函数的高亮支持；Python 常量语义高亮得到改进
  • 支持 textDocument/semanticTokens 能力的动态注册
  • 协作面板新增自动观看按钮，可在通话时自动打开其他参与者的屏幕共享
  • Vim/Helix 模式新增 z c 居中滚动及 alt-b/alt-e 语法节点跳转等快捷键
  • 新增 Markdown 预览字体大小独立设置及项目面板右键打开预览功能
  • 优化文件监听器事件批处理，提升高文件系统负载下的响应速度
  • 改进文本查找器，支持在匹配列打开文件并记忆上次查询与过滤器
  • 项目面板右键菜单新增全部展开 / 折叠选项及快捷键
  • 提升网络韧性，通过检测并丢弃陈旧 HTTP 连接来减少间歇性网络错误
  • Windows 平台改进代理沙箱，增强对 Git 元数据的保护

Breaking Changes

  • 默认禁用保存时格式化 (format-on-save)，仅对具有官方格式化程序的语言保持开启
  • LLM 提供商、外部代理和 MCP 服务器的配置现已迁移至统一的设置编辑器中

问题修复

  • 修复 Linux 终端进程跟踪导致的文件描述符泄漏及剪贴板失效问题
  • 修复 Git 仓库在文件系统强制重新扫描时被重复创建导致 UI 抖动的问题
  • 修复 MCP 服务器特定工具名称在 Anthropic/OpenAI 模型中导致错误的问题
  • 修复远程会话无法识别仅语言扩展提供的语言问题
  • 修复终端中右键与中键互换的问题
  • 修复 Markdown 表格在包含管道符的行内代码段中的渲染问题
  • 修复远程主机路径风格不同导致 “信任文件夹” 输入拒绝有效路径的问题
  • 修复多种 Code Lens 相关问题（动态注册不刷新、远程工作流不解析、折叠区域不隐藏）
  • 修复 Helix 模式在符号最后一个字符处重命名的 Bug
  • 修复远程项目重连或重启时 Git 提交消息被清除的问题
  • 修复 Linux Wayland 鼠标指针卡在加载状态的问题
  • 修复终端 copy_on_select 在点击抖动时覆盖剪贴板的问题

steipete/CodexBar v0.45.0 - v0.45.1 gpt-oss-120b

新内容

  • 新增 Claude 每模型每周使用窗口显示（来自 schema-v1 账户列表）
  • 新增 Claude 可选卡片，仅在单一账户时显示
  • 新增 OpenCode Go 每日本地成本和计划使用历史
  • 新增概览将合并提供者上限从 3 提升至 6
  • 菜单栏移除状态项悬停提示，保持 VoiceOver 标题
  • Codex 成本标签简化为 “Cost”，并将报告与估算说明迁入成本设置
  • 菜单栏新增拖拽布局，可自定义身份、使用、重置、费用、间距及堆叠代币。
  • CLI 新增配额感知守护、浏览器 Cookie 刷新、稳定退出码、JSON 输出等功能。
  • 新增组织 API‑key 本地费用估算、Cursor 代币成本报告。
  • 新增 OpenRouter 多账户隔离、DeepInfra 余额与限制、DeepSeek 成本与代币历史。
  • 其他新增：OpenCode 安全 Cookie 重导入、适配式自适应模式、代理会话标签、30 天组织支出统计、Doubao 计划使用…

问题修复

  • 修复 StepFun 登录时 Web ID 派生错误，确保头部和 Cookie 与匿名令牌匹配
  • 修复菜单栏刷新、使用量预测、成本 / 令牌指标对齐、Codex 子代理统计、分享卡 PNG 导出、模型行显示及 ElevenLabs 使用百分比超限等问题
  • 修复菜单栏调色板拖拽及布局编辑器焦点环问题，确保 Pill 正确粘贴并保留键盘访问。
  • 修复提供者版本探测，仅对已启用提供者进行，防止禁用提供者启动子进程或触发 TCC 提示。
  • 修复 Linux CLI 长时间服务进程的管道泄漏导致 EMFILE 崩溃。
  • 修复 Claude CLI 在解析错误后保留上一次成功使用并重用探测会话，避免创建空账户会话。
  • 修复浏览器会话重试逻辑，防止旧 Cookie 掩盖活跃 Vivaldi 会话。
  • 修复 Widget 刷新节奏对齐并加入 5 分钟安全间隔。
  • 修复 Cursor 与 Abacus 超额使用限制，计划使用和信用使用均上限 100%。
  • 内部测试迁移至任务本地范围并清除死代码，强化自适应扫描限制。

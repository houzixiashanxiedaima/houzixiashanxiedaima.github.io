anomalyco/opencode v1.17.10 - v1.17.11 GPT-OSS 120B

新内容

  • 添加会话快照和回滚控制，可将会话回滚到更早的消息和文件更改。
  • 添加 Chrome 风格的标签循环快捷键（mod+1~mod+9）。
  • 支持可拖拽的标签页。
  • 改进空主页状态，当没有会话时提供提示。
  • 优化桌面布局，改进标题栏标签和归档会话的显示。
  • 新增 MCP 服务器指令到会话上下文
  • 新增 —mini CLI 模式
  • 新增可配置快捷键打开差异查看器
  • 新增移动端底部导航和可折叠服务器列表
  • 在 SDK 中公开提供者集成 ID 并支持 V2 插件 API

问题修复

  • 始终打印 MCP OAuth URL，确保手动登录在浏览器流程中仍可使用。
  • 修复提示草稿未正确关联到对应项目服务器的问题。
  • 恢复标题栏标签大小和布局的一致性。
  • 当会话列表为空或加载中时，恢复主页操作按钮。
  • 自动关闭已不存在会话的标签页。
  • 隐藏无权限时的 MCP 资源模板工具
  • 绑定 MCP OAuth 回调到 IPv4 回环，提高本地认证可靠性
  • 修复跨平台路径导致的 ACP 资源文本读取问题
  • 防止工作线程拒绝导致 TUI 进程崩溃
  • 持久化草稿和提示状态更可靠，提升会话导航稳定性

jdx/mise v2026.6.14 GPT-OSS 120B

新内容

  • 新增 bootstrap —only 参数，可与 —skip 互斥，支持逗号分隔阶段名
  • 新增 bootstrap packages import /prune，支持 Homebrew 配
  • 新增 bootstrap status（别名 ls），聚合多类资源并支持 —json 与 —mi
  • brew‑cask 安装器现支持二进制 cask 包，可在 Caskroom 暂存并链接到 Home
  • 后端资产平台检测新增对 riscv64 与 loongarch64 的识别，确保相应工具正确安装

问题修复

  • which 自动启用 prefer_offline，满足已安装版本时不再发起远程 HTTP 请求
  • sigstore TUF 根获取遵循 settings.url_replacements，修复镜像环
  • config 将符号链接的全局配置路径视为全局，避免在 Fedora Atomic 上被误判
  • mise set 等 TOML 写入定位最近的配置目录，防止嵌套项目误更新父级 mise.toml
  • 对只读版本文件的写入尝试返回错误而非 panic，提高稳定性

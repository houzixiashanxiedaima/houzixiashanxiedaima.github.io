steipete/CodexBar v0.44.0 gpt-oss-120b

新内容

  • 新增多家提供商的使用统计功能：ZenMux、ClinePass、LongCat、Neuralwatt、Codex、Cursor、Copilot、Grok…
  • 新增 Settings 本地使用与支出视图，提供 7/30 天覆盖、原生货币分组及精确的 Codex 账户扫描
  • 新增 Settings 私有本地分享卡，显示原生货币总计并对计划 / 模型标签进行脱敏
  • 新增 Hooks 可选外部命令，响应配额及提供商状态变化，具备无 shell 执行和刷新风暴保护
  • 新增 CLI token‑gate 的仪表盘快照功能，敏感响应不缓存，LAN 绑定需显式 HTTP 选项
  • 新增 CLI 可选 claude‑swap 账户卡片，支持完整与简略使用展示并保留账户与来源覆盖
  • 新增菜单栏高对比模式，可在非活动显示器上保持图标与百分比可读性
  • 新增 MiMo 功能，可从受限会话恢复文件中恢复仅会话的 Firefox Cookie
  • 新增 Confetti 使用提供商调色板进行重置庆祝

Breaking Changes

  • 移除 Kimi K2 非官方匿名中继，保留官方 Kimi 与 Moonshot 支持
  • 移除 CrossModel 托管中继，需注意相关操作将不再可用

问题修复

  • 修复 Codex、Claude、Cursor 等计数与配额显示错误，包括子代理计数、周配额隐藏、USD 总计标签等
  • 修复 CLI 与 RPC 输出缓冲导致的内存增长问题
  • 修复浏览器 Cookie 访问限制、StepFun 信用计划显示错误以及 Ollama Safari 完整磁盘访问提示
  • 修复命令代码计费链接指向错误、工作流脚本注入风险以及内部测试隔离问题
  • 修复 Antigravity 配额优先设置、Ollama API‑key 验证提示、Claude 重复配额行、Copilot 配额条隐藏等若干细节

code-yeongyu/oh-my-openagent v4.19.0 gpt-oss-120b

新内容

  • 新增对 Kimi K3 模型的一级支持，包括模型检测、回退路由以及针对 Sisyphus、Sisyphus-Junior、Atlas 的专属提示变体
  • 替换旧的 Ralph Loop，推出全新 Goal 功能，具备状态持久化、解析器、配置迁移及 /goal 命令
  • Senpi 任务引擎可靠性提升：任务丢失时正确释放、TTL 清理、文件描述符复用等

  • 支持 Playwright MCP 自定义参数、模型核心检测器等功能改进

Breaking Changes

  • 移除 Ralph Loop 接口，用户需改用 Goal 功能并迁移配置
  • 重命名多处包检测、插件 / 配置兼容性以及安装入口，可能导致升级时行为变化
  • 删除共享技能别名及保护机制，相关自定义技能需相应调整

问题修复

  • 修复 Codex V2 线程容量保持、ulw‑plan 解析校验及代码图 git‑marker 处理安全性
  • 解决任务引擎、TTL、记录存储等导致的长时间多代理工作流不稳定问题
  • 合并多项文档与测试修复、依赖升级、重复技能删除等琐碎修复

  • 文件上传端点路径遍历硬化

  • SQL 控制台默认禁用，需在 config.ini 中手动启用

  • OPML v1 导出已移除，仅保留 v2，v1 仍可导入

问题修复

  • 修复移动端上传附件失败

  • 修复空笔记查看版本差异时显示 “Diff 不可用”
  • 修复代码块输入 @ 时错误创建笔记并提供斜杠命令提示
  • 修复长时间离线后同步中断导致的笔记恢复问题
  • 其他细节改进与 UI 稳定性提升

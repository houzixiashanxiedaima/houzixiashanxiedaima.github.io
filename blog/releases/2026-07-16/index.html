ogulcancelik/herdr v0.7.4 gemma-4-31b

  • 新增对 Maki agent 的支持
  • 新增可搜索的版本匹配配置参考和故障排除指南
  • 扩展了日语和简体中文的 CLI 文档，包含 Shell 补全设置和 API 模式用法
  • 更新内置的 Herdr agent 技能以适配当前的公共工作区和 API 工作流

Breaking Changes

  • 扩展的 Space 和 Agent 侧边栏条目现在默认使用紧凑布局（可通过将 row_gap 设为 1 恢复）

问题修复

  • 修复 Agent 侧边栏折叠行排序、点击目标及快捷键编号分配问题
  • 修复插件驱动的标签重命名无法立即刷新标签栏几何形状和标签的问题
  • 修复新标签 / 拆分 / 工作区在跟随前台目录时，未从当前聚焦面板目录启动的问题
  • 修复对 Amp、Codex 和 Claude Code 的检测，支持最新的 UI 变体
  • 修复 Pi 生命周期状态在原生会话替换后未重新锚定的问题
  • 修复 WSL 默认光标显示，解决 Windows 宿主光标闪烁问题
  • 修复实时移交（Live handoff）过程中 Socket 路径保留和服务器关闭同步问题
  • 修复 Windows 安装程序在首次运行时覆盖配置或创建重复引导行的问题
  • 修复分离的自定义命令子进程在退出后产生僵尸进程的问题
  • 修复 Linux 前台进程发现机制，限制在 Herdr 进程树内以降低 CPU 占用
  • 修复 Windows 原生服务器未脱离启动终端导致关闭终端即停止进程的问题
  • 修复 Windows API 客户端在等待初始请求时出现 BrokenPipe 的问题
  • 修复 herdr —remote 在非 POSIX 登录 Shell（如 xonsh）上的安装问题
  • 其他修复：包括 Shift 索引绑定、OMP 生命周期报告、配置诊断、标签可见性、文档搜索范围、鼠标滚轮 / 触控板事件、复制模式边界跳转、SGR 鼠标报告…

steipete/CodexBar v0.39.0 gpt-oss-120b

新内容

  • 新增在菜单和设置中显示 Codex 所有可用的信用到期信息，并汇总即将到期的信用
  • 新增成本历史对比，可选 7、30、90 天的短期比较
  • 新增 Codex 成本历史，按项目和工作树在菜单和 CLI 中分组显示

  • 新增 Kimi 每月订阅使用量及周 / 五小时限制并显示总预算

  • 修复 Claude 在模型范围内的每周和每日配额阈值通知
  • 修复 Claude CLI 在终端错误或加载卡顿后跳过身份探测，减少刷新延迟

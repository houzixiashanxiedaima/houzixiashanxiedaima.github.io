code-yeongyu/oh-my-openagent v4.9.0 - v4.9.1 GPT-OSS 120B

新内容

  • 新增共享 per-user LSP 守护进程，实现跨会话语言服务器复用
  • 新增 lsp-setup 技能，提供 20+ 语言服务器的检测、安装、配置与验证
  • 新增内置 Julia 与 Razor 语言服务器支持
  • 引入 Node CLI 运行时回退，om​o 不再强制依赖 Bun
  • 改进 Sparkshell 输出摘要，支持 per-model 提示变体并使用真实 gpt-5.3-

问题修复

  • 修复 LazyCodex (Codex Light) 在 v4 版本的崩溃问题
  • 修复 zero‑dependency 环境下 omo doctor 的崩溃
  • 失败的工具调用不再导致长驻代理崩溃
  • 重试机制避免请求重复执行
  • 瞬时故障不再污染会话可用性缓存
  • Codex 安装程序在更新时清理过期守护进程

Breaking Changes

  • LSP 不再在进程内运行语言服务器，改为通过共享守护进程的代理
  • multi_agent_v2 在所有 Codex 会话启动时被强制禁用，配置自动写入 disable
  • Sparkshell 摘要默认模型更改为 gpt-5.3-codex-spark，且不再隐藏秘密
  • 移除 keyword-detector 的 search 与 analyze 模式以及 anthro
  • experimental.aggressive_truncation 选项现已生效，之前始终开启

HandBrake/HandBrake 1.11.2 GPT-OSS 120B

新内容

  • 更新 FFmpeg 至 8.0.2，提升解码和滤镜功能
  • 更新 SVT‑AV1 至 4.1.0，支持 AV1 编码
  • 在 Linux 添加 WebM MIME 类型，支持 WebM 格式
  • 更新音频抖动器和编码器组合列表
  • 改进 Mac 与 Windows 对不支持的预设的处理

问题修复

  • 修复 2‑pass 无损 x265 编码时的崩溃
  • 修复 2‑pass MPEG‑4/MPEG‑2/VP9/FFV1 编码的内存泄漏
  • 修复 Core Audio AAC 编码的 7.1 声道布局
  • 修复 MP4 容器中 VobSub 调色板创建错误
  • 修复 Windows 自动音轨名称生成问题

Breaking Changes

  • 升级后自定义预设和偏好可能不兼容，需提前备份
  • Windows 需要安装 .NET Desktop Runtime 10 才能运行

  • 修复 Battery 弹窗中顶部进程高度计算错误
  • 将最低 macOS 版本调整至 12.0，提升兼容性

fishjar/kiss-translator v2.0.23 GPT-OSS 120B

新内容

  • 优化字幕翻译的流式渲染、AI 断句和错误处理
  • OpenAI 兼容接口支持推理模式配置
  • 重构 YouTube 字幕提供器结构，拆分字幕轨道等模块
  • 优化 TranslatorManager 在 SPA 页面中的生命周期处理

问题修复

  • 修复已处理节点在 SPA 页面中被重新扫描的问题
  • 修复划词翻译手动选择后被自动切换的错误
  • 修复 popup 中无害标签页消息错误
  • 修复油猴脚本环境变量读取问题
  • 修复 Origin 头伪装范围和右键菜单文案问题

Breaking Changes

  • 移除 iframe 白名单设置，空 iframe 将不再触发翻译

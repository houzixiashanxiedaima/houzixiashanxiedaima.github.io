sst/opencode v1.0.153 Qwen 2.5

新内容

  • 添加 ARM64 Docker 镜像支持
  • 添加 Gleam LSP 和格式化工具
  • 添加 TexLab 语言服务器和 latexindent 格式化工具
  • 添加 Dockerfile 语言服务器
  • 添加 —session 标志以附加命令

问题修复

  • 修复透明主题下输入边框未绘制的问题
  • 修复 TUI 在继续标志时打开父会话而非子代理的问题

  • 添加服务器错误、速率限制和无 KV 空间重试逻辑以适应 Foundry API 问题

jxxghp/MoviePilot v2.13.11 GPT-OSS 120B

新内容

  • 新增可配置的 Redis 客户端连接池设置
  • 支持安装发布版本
  • 持久化代理聊天历史并可通过 save_history 标志控制保存
  • 为 Telegram 机器人添加文字编辑失败时的字幕编辑回退
  • 增强 WebAgent 音频处理，支持格式转换和转录

问题修复

  • 优化发布缓存加载
  • 排除通知历史标志在事件负载中的传递
  • 修复工作流过滤输出为空时的保留问题
  • 改进 append_text 文本处理并确保换行符保存
  • 修复语音上传文件句柄及 Content-Type 问题

tw93/Pake V3.11.10 GPT-OSS 120B

新内容

  • 新增 Arch Linux .zst 包格式，适配 Arch 系发行版
  • 为 pake-cli GitHub Action 添加 —min-width、—min-heig
  • AppImage 构建失败时提供更清晰的 sudo 步骤提示

问题修复

  • 修复 macOS 认证弹窗可能出现的崩溃
  • 修复 niri Wayland 合成器下 AppImage 的输入处理
  • 修复 Linux 上标题栏装饰隐藏并保留显示标题

Breaking Changes

  • 许可证从 MIT 改为 GPL-3.0，需遵守新许可证
  • CLI 参数校验更严格，拒绝空值和非法数值

anomalyco/opencode v1.17.8 GPT-OSS 120B

新内容

  • 会话时间线加载更快且避免闪烁或滚动跳动
  • 新增首页标签切换按钮，可快速在首页和上一次标签间切换
  • 新版桌面文件 / 文件夹选择器在 v2 布局下更快捷易用

问题修复

  • OpenAI 兼容的提供商现在接受之前验证失败的 MCP 工具模式
  • Cloudflare AI Gateway 正确接收配置的 API 密钥
  • 长运行的 MCP 工具在报告进度时保持超时存活
  • MCP OAuth 回调服务器在授权完成或取消后自动关闭
  • MCP 工具错误现在显示服务器的错误文本而非通用失败信息

code-yeongyu/oh-my-openagent v4.11.1 GPT-OSS 120B

新内容

  • Codex 采用原生 Sisyphus 路径运行
  • LazyCodex 市场安装保持升级和钩子重新批准边界
  • CodeGraph MCP 路径在首次会话前完成规范化
  • 共享模型检测更可靠地处理最新提供者命名变体
  • 编程技能将超过 250 行代码视为硬架构缺陷

问题修复

  • 修复 Codex 市场自动更新边界问题
  • 修复 CodeGraph MCP 路径在引导前未规范化的问题
  • 修复 GPT-5.3 Codex 路由为原生 Sisyphus 的问题
  • 修复编程技能的硬 LOC 门限恢复
  • 修复模型核心版本分隔符的规范化

Breaking Changes

  • 重命名跨包检测、插件 / 配置兼容性和安装入口的过渡
  • 兼容性行为变更可能影响现有插件

rorkai/App-Store-Connect-CLI 2.1.1 GPT-OSS 120B

新内容

  • 新增匿名 CLI 遥测控制
  • 新增 iOS 代理发布指南文档
  • 在应用墙中新增 Tiny Decisions

firebase/firebase-tools v15.21.0 GPT-OSS 120B

新内容

  • Functions 可声明额外的 API 依赖
  • 在 Auth 模拟器中新增 mock Passkey（WebAuthn）支持
  • Firebase Console 现已支持 Dart Functions，移除旧警告
  • Firebase Data Connect 本地工具升级至 v3.4.11
  • Web 框架在 Angular 22 部署时提示缺失 SSRF 防护配置

问题修复

  • 修复 firebase —help 中缺失 login、target、ext 子命令
  • 解决 Windows 初始化 Python Functions 时 spawn activate.b
  • 修正 Hosting 模拟器的 Cloud Run 重写始终调用线上 API 的问题
  • 修复 apptesting:execute 命令不接受 —test-username 等选项的错误
  • 修复 Astro 6 SSR 部署返回 HTTP 500 的问题

lwouis/alt-tab-macos v11.3.1 GPT-OSS 120B

新内容

  • 避免在主线程进行阻塞调用的性能提升

问题修复

  • 解决 macOS 睡眠 / 唤醒导致的错误
  • 修复跨空间聚焦时的副作用
  • 解决极少出现的 tooltip 导致的崩溃
  • 防止使用 Escape 键触发切换器时的问题
  • 隐藏 Android 模拟器空标题窗口

google-gemini/gemini-cli v0.47.0 GPT-OSS 120B

新内容

  • 支持后端定义的 3.5 flash，并在启用标志时自动使用

  • 为 Antigravity CLI 添加文档和迁移命令

问题修复

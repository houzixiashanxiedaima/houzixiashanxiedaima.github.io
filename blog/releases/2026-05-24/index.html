code-yeongyu/oh-my-openagent v4.4.0 Qwen3 235B (Cerebras)

新内容

  • 新增安全研究团队模式技能，支持并行漏洞挖掘与 PoC 验证
  • 团队模式基于实际可利用性评估漏洞，集成 CWE、OWASP 与 CVSS v4 标准
  • 支持 0 信任评分机制，需启用 team_mode 并配置 enabled: true
  • TUI 支持点击子代理条目，修复延迟会话 ID 问题
  • look_at 命令不再卡住，无输出时优雅退出

问题修复

  • 修复父唤醒竞争导致的重复助手流问题
  • 模型解析失败时仍保留 Atlas 模型覆盖设置
  • multimodal-looker 现信任用户指定的视觉模型
  • 阻止 gpt-5.3-codex 被错误重写为 gpt-5.4
  • 修复 Windows 路径匹配及 CJK 终端标题乱码问题

Breaking Changes

  • 移除顶层 lsp 配置项，请迁移至 opencode/lsp 配置

  • 修复 V2Ray 链接和 v2rayN 导入时 XHTTP 字段解析问题
  • 修复移动端表单字段堆叠显示问题

Breaking Changes

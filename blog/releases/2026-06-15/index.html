jdx/mise v2026.6.8 - v2026.6.10 GPT-OSS 120B

新内容

  • 新增 env 默认快捷方式（default = “fallback”），仅在变量未设置或为空时生效
  • .7z 压缩包在 Unix 系统上支持解压，并实现跨平台 strip_components 探测
  • 后端库迁移至 sevenz-rust2，强化安全性，拒绝目录遍历和绝对路径
  • 文档站点头部社交图标垂直居中

问题修复

  • env_shell_expand 启用时，${VAR:-default} 按 POSIX 语义处理空
  • mise x/run/env 在子进程环境中剥离 mise 管理的安装目录，避免 PATH 中残留
  • vfox 插件安装钩子在 Lua os.execute 时使用统一的 sanitized mise_
  • Windows 无扩展名的 Bash shim 检测 WSL 并直接执行工具，防止无限循环
  • Unix make_symlink 采用临时文件再原子 rename，消除 “File exists
  • ruby 预编译安装必须使用数字构建版本标签，忽略浮动基线版本
  • zig@master 现作为滚动发布渠道，解析为具体 nightly 版本并安装到版本化目录

ccusage/ccusage v20.0.12 - v20.0.13 GPT-OSS 120B

新内容

  • 新增在 arm64 runner 上交叉构建 Intel macOS 二进制
  • 修复 update-pricing 工作流失败
  • 使用 cache-nix-action 缓存 Rust 构建
  • 将嵌入快照缩小 58%，二进制体积缩小 2.9%
  • 支持在不使用 Nix 开发环境的情况下发布

问题修复

  • 修复 pricing 模块中避免显示模型别名的慢回退问题
  • 在缺少系统 tz 数据库的平台上打包 tz 数据库

anomalyco/opencode v1.17.7 GPT-OSS 120B

新内容

  • MCP 服务器现在可以接收当前工作区作为客户端根目录
  • SDK 客户端在集成更改时刷新模型和提供者可用性，凭证更新 / 删除调用支持位置

问题修复

  • 插件客户端请求现在复用活动服务器，而不再默认本地端口
  • ACP shell 工具调用现在从一开始就显示命令和工作目录
  • 插件提供的 shell 环境变量现已在 PTY 会话中生效
  • MCP 调试现在使用 SDK 的最新协议版本
  • 新的会话路由保持在其草稿服务器范围内，确保提示和状态指向正确工作区

  • 清理过期 tmux 窗格并改进缺失文件错误提示
  • 修复唤醒和团队交付相关的错误

Breaking Changes

  • 任务系统默认行为对齐，省略配置时行为统一

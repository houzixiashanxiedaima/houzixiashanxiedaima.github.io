NousResearch/hermes-agent v2026.7.1 gpt-oss-120b

新内容

  • Mixture-of-Agents 现已成为一等模型，可直接在模型选择器中选择
  • 支持在 MoA 集成时显示每个模型的完整推理并标记
  • 代理能够自行验证工作完成情况，记录验证证据
  • 新增 /learn 命令，可将目录、URL 或工作流转化为可复用技能

问题修复

  • 已关闭所有 P0 级别问题，实现零未解决 P0
  • 已关闭所有 P1 级别问题，实现零未解决 P1
  • 解决了 interrupt‑protected‑compression fork 的关键缺陷
  • 加强了凭证泄露防护（credential‑exfil hardening）
  • 提升了 cron 可靠性，修复相关问题

zed-industries/zed v1.9.0 gpt-oss-120b

新内容

  • 新增 Telescope 风格可调大小的选择器，支持文件和文本查找预览并实时显示搜索结果。
  • 在 AI 面板中加入线程内搜索（Ctrl/Cmd+F）及快速添加远程 MCP 服务器的选项。
  • 为 Git 面板添加分割按钮的变更头、Git blame 切换、视图选项菜单以及默认点击行为设置。
  • 为终端 Vim 模式加入段落导航快捷键（Shift-{/ Shift-}），并在 Helix 模式
  • 新增 markdown_preview.limit_content_width 与 markdown

sindresorhus/Gifski v3.0.3 gpt-oss-120b

问题修复

  • 修复了对带透明度视频的处理

lwouis/alt-tab-macos v11.4.0 gpt-oss-120b

新内容

  • 使用 WindowServer 事件替代 Accessibility
  • 在异常列表中仍显示活动窗口
  • 在 macOS 未保存偏好时给出警告

问题修复

  • 修复 macOS 26 上构建设置窗口的崩溃
  • 修复通过 Universal Control 的触控板手势导致的崩溃
  • 修复自定义面板在深浅模式切换时未实时更新
  • 修复标签页未显示为独立窗口的回归问题

immich-app/immich v3.0.0 gpt-oss-120b

新内容

  • 移动端支持非破坏性编辑，可直接裁剪、旋转、调节图片
  • 移动编辑采用与网页相同的编辑系统，实现原图不变
  • 工作流预览功能上线，可通过拖拽构建自动化流程
  • 工作流编辑器支持可视化和 JSON 两种模式切换

Breaking Changes

  • API 端点更新，第三方工具需根据迁移指南调整
  • 不再支持 pgvecto，相关集成需改用其他方案
  • 运行在 v1 之前的 Immich 需要迁移至新版本

hugohe3/ppt-master v2.13.0 gpt-oss-120b

新内容

  • 支持导出时保留动画图片，检测多帧图像并原样传递

  • 项目初始化时避免重复后缀，已存在相同后缀的名称直接使用
  • source-to-md 支持批量转换，可一次处理文件、目录和 URL，网页按标题命名
  • Overlay & Masking 新增切片位移规则，文本必须位于不透明平面上

问题修复

  • 修复 v2.12.0 回归导致动画 GIF 在导出时被截帧的问题
  • 修复项目初始化时双后缀错误
  • 修复路径 B 主机工具生成错误分辨率并防止上采样
  • 修复跨版本引用失效的问题并完善文档
  • 完成最后一个 CLI 脚本的 UTF-8 控制台编码支持

nextai-translator/nextai-translator v0.6.29 - v0.6.31 gpt-oss-120b

新内容

  • 性能优化，显著降低 UI 卡顿
  • 输入框键入时不再无谓重新计算高亮覆盖层
  • 原生模糊层不再在主题切换时堆叠，遵循窗口圆角
  • OCR 识别不再静默失败，捕获后始终弹出翻译窗口
  • OCR 选区矩形限制在截图范围内，解决缩放显示下随机失败
  • Edge TTS 已下线，自动切换至本地 TTS，未支持语言回退至系统语音
  • 历史窗口的关闭 / 清除按钮固定在底部
  • 更新器窗口滚动正常，长发布说明可完整阅读

问题修复

  • 修复 OCR 工作线程在图像被拒绝后泄漏导致 UI 逐渐卡顿
  • 修复语音合成失败后泄漏的音频管道，防止后台 CPU 持续占用
  • 修复 OCR 识别失败时加载指示器卡住的问题
  • 修复窗口左上角灰色伪影，避免背景模糊时出现

jdx/mise v2026.7.12 gpt-oss-120b

新内容

  • 新增安全模式 MISE_SAFE=1，阻止配置执行代码和环境注入
  • 新增 mise lock —bump，支持在不修改 mise.toml 的情况下更新锁文件
  • 新增无 Node 的 npm 后端，直接通过 HTTP 解析并在进程内安装包
  • 内置 OCI 注册表客户端，取代外部 skopeo/crane
  • OCI 推送支持分块上传、跨仓库挂载、进度显示等增强
  • OCI 推送实现层复用，跳过未变更层的构建
  • 支持多架构 OCI 镜像索引更新
  • 内置签名验证，使用 Rust rPGP 替代外部 gpg
  • bootstrap.repos 支持相对路径，提升项目可移植性

Breaking Changes

  • 移除 mise oci push —tool，需改用 mise oci build -o ./img 再使用 skopeo copy
  • 启用 Node/Swift 签名验证后始终执行，若依赖缺失 gpg 跳过验证需设置 gpg_verify = false
  • npm.use_npm_view 设置被 npm.shell_out（MISE_NPM_SHELL_OUT）取代

问题修复

  • 修复 prefer_offline 超时回归及相关网络请求超时问题
  • 修复 lock 跨平台时错误修剪 OS 限制工具条目
  • 修复 env 文件解析、变量引用及 Windows _.source 行为
  • 修复 config 相关错误：trusted_config_paths、数组解析、set panic、semver 警告
  • 修复多项平台和工具兼容性问题（zsh、brew、backend、completions、link、shim、vfox…

zed-industries/zed v1.12.0 gemma-4-31b

新内容

  • Git 面板新增暂存（Staged）与未暂存（Unstaged）分组，并支持快速切换状态及丢弃更改
  • 文件查找器与文本查找器支持多选，可通过快捷键或按钮一次性打开多个项目
  • AI 增强：支持自定义 Anthropic 模型的自适应思考与快速模式，新增 GPT 5.6 系列及 Grok 4.5 模型
  • AI 增强：默认启用 ACP 引导，支持扩展 MCP 工具调用详情，改进技能删除流程
  • 语言支持：新增可过滤且带预览的结果选择器（查找引用、转到定义 / 实现）
  • 支持在 Markdown 文件中粘贴剪贴板图片，并优化预览与源码的切换快捷键
  • 新增 format_on_save 的修改模式，支持仅格式化 Git 变更行以减少 diff 抖动
  • 新增编辑器缩放切换动作，可最大化活动编辑器面板
  • 支持将 gitignored 子目录作为独立工作区打开，并支持减少动画设置
  • 支持在项目搜索中使用正则表达式子例程调用

问题修复

  • 修复多种崩溃问题，包括 Linux Wayland 文件对话框、Git Graph 及 macOS 窗口管理崩溃
  • 修复 AI Agent 在 LSP 无响应时挂起以及 edit_file 工具损坏缩进的问题
  • 修复 Git GPG 签名在缓存密码时重复提示的问题
  • 修复 Python 虚拟环境在重新打开工作区时无法自动恢复的问题
  • 修复 Wayland 环境下 IME 提交导致剪贴板失效的问题
  • 修复多种 UI 渲染问题，包括内联提示选择、Markdown 预览闪烁及终端闪烁
  • 修复非可写文件被错误地在可编辑缓冲区中打开的问题
  • 修复 pnpm 工作区文件保存后路径变为符号链接的问题
  • 修复多种选择器（Picker）和面板（Agent Panel）的交互 Bug 及 UI 挂起问题

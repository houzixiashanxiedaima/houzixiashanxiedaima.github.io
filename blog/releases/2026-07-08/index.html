mattpocock/skills v1.1.0 gemma-4-31b

  • 优化 to-tickets 技能，支持处理大规模重构并适配本地文件或真实追踪器
  • 正式发布 wayfinder 技能（原 decision-mapping），用于规划大规模工作流
  • 扩展 triage 技能，支持对外部 Pull Request 进行分拣处理
  • 将 tdd 技能重塑为参考类技能，引入 seam 概念并增加同义测试反模式

Breaking Changes

  • decision-mapping 重命名为 wayfinder

jdx/mise v2026.6.13 GPT-OSS 120B

新内容

  • 新增 brew cask 包支持，支持 sudo 安装并追踪 pkgutil receipt ID
  • 注册表新增 lore 并实现对多种工具的已发布校验和解析

问题修复

  • mise lock 在无法解析活跃工具请求时报错，避免静默跳过并更新 lockfile
  • aqua 支持在校验和算法不同的情况下升级已记录的校验和
  • task 运行和 hook‑env 采用离线解析构建初始工具集，避免启动时触发远程 GitHub 查
  • node 安装记录各平台结果及源 tarball URL 与校验和，避免写入错误的二进制 URL
  • Windows 文件插件使用目录符号链接代替 junction，插件列表可显示并删除

  • 开源版新增 AI Relay、Video Relay 与 Relay Media 支持

  • 修复本地 Docker 自动登录在前端清空登录态后无法恢复的问题

Breaking Changes

  • Relay 配置已迁移至配置管理界面，需使用 UI 或 config.yaml 替代手动编辑 doc
  • Docker 部署方式改为挂载 config.yaml，旧的 docker-compose.yml

fastfetch-cli/fastfetch 2.65.1 GPT-OSS 120B

新内容

  • 新增对 x86 CPU 代号和制造工艺的检测，可通过 {code-name}、{technology
  • 新增对 GPU PCIe 链路速度（当前和最高代数及通道）的检测，可通过 {pcie-max-spe
  • 新增通过 Wayland wp-color-management-v1 协议的 HDR 显示检测，已

  • 修复 Codec 模块在开启 codec.showType 时错误返回无结果的情况。

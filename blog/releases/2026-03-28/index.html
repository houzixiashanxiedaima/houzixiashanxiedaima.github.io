groupultra/telegram-search v1.2.1 Qwen3.5 27B

新内容

  • 移除不必要的日志
  • 新增账户设置缓存

beck-8/subs-check v1.4.9 Qwen3.5 35B

新内容

  • 将单一 worker 池拆分为测活、测速、流媒体与重命名三个独立阶段
  • 新增 speed-concurrent 和 media-concurrent 配置项，支持独立并发控
  • 测速与流媒体检测改为并行执行，避免带宽争抢和资源冲突
  • 全新视觉风格，新增三阶段步骤指示器实时显示当前检测进度
  • 左右面板底部对齐，优化整体布局协调性

问题修复

  • 修复获取 IP 位置和流媒体串行导致最后节点卡住不结束的问题

fastfetch-cli/fastfetch 2.61.0 Qwen3.5 27B

新内容

  • 新增颜色显示配置的亮度选项
  • 新增 Linux 蓝牙键盘检测支持
  • 新增显示 Linux 空内存插槽的选项
  • 新增对新款 M5 Mac 机型的支持
  • 改进 SMBIOS 鲁棒性及 Intel Mac 支持

问题修复

  • 修复某些机器上缺失内存设备的问题
  • 修复共享缓存的 CPU 缓存去重问题
  • 修复 niri 窗口管理器的版本报告问题
  • 修复来自 iw 输出的 SSID 解码问题
  • 修复在 Windows 上运行后 CMD 代码页被更改的问题

Zephyruso/zashboard v2.9.0 Qwen3.5 27B

新内容

  • 新增 Apple 风格主题，UI 优化至 macOS/iOS 美学
  • 增强设置组件，改进对话框处理与网格布局
  • 优化设置页面滚动跟踪，调整刷新按钮位置
  • 优化概览图表，引入 MiniSparkline 并改进轴标签
  • 重组设置布局，整合面板样式部分

rikkahub/rikkahub 2.1.7 Qwen3.5 27B

新内容

  • 改进 Key 轮询策略为 LRU 轮询
  • 新增 Tavily 搜索结果摘要，Exa 搜索支持 deep 模式
  • 新增支持 GLM-5.1 模型
  • 新增音量键翻页功能，适配墨水屏设备
  • 通知标题显示当前模型或助手名称

问题修复

  • 修复历史对话打开时无法自动滚动到底部的问题
  • 修复 Word 文档解析内容为空的问题

yikart/AiToEarn v2.1.0 Qwen3.5 27B

新内容

  • 新增 OpenClaw（龙虾）支持，可在其中直接使用 AiToEarn
  • 新增 MCP 协议支持，兼容 Claude、Cursor 等支持 MCP 的 Agent

问题修复

  • 修复大量可用性问题

google-gemini/gemini-cli v0.35.3 Qwen3.5 35B

问题修复

  • 修复：处理合并冲突并应用提交 765fb67

wavetermdev/waveterm v0.14.4 Qwen3.5 35B

新内容

  • 新增垂直标签栏选项，支持沿窗口侧边显示标签页。
  • 升级 xterm.js 至 v6.0.0，改善终端兼容性与渲染效果。
  • 文件流传输迁移至新接口，修复首字节延迟严重问题。
  • 配置错误移出标签栏移至设置视图，减少界面杂乱。
  • WaveConfig 视图增加未保存更改前的警告提示。

问题修复

  • 修复新建块时的焦点丢失问题。
  • 修复拆分旧会话启动新持久会话的问题。
  • 修复 Electron v41 回归导致的 webContents 为空问题。
  • 优化 macOS 首次点击处理，正确设置 WaveAI 焦点。
  • 修复预览环境中 PDF 显示异常问题。

Breaking Changes

  • 已移除弃用的旧版 AI 组件，请停止使用相关功能。

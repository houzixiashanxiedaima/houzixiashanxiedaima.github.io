xishang0128/sparkle 1.26.6 GPT-OSS 120B

新内容

  • 新增多架构 Linux 安装包（64 位、ARM64、Apple Silicon、loong64）
  • 更新 nupdate 依赖以提升兼容性

问题修复

  • 修复 proxy item subtitle 中 descender 被裁剪的问题

anomalyco/opencode v1.17.12 - v1.17.13 gpt-oss-120b

新内容

  • 改进 markdown 和消息布局对齐在 v2 会话 UI
  • 添加可搜索的 v2 模型选择器和模型管理流程
  • 在会话标签上悬停预览项目、路径、分支和服务器信息
  • 简化 WSL 服务器设置，提供更清晰的发行版检查和安装流程
  • 将桌面标签范围限制在每个窗口，使窗口恢复各自的标签集
  • 新增 yolo 模式，实现权限自动批准
  • 在编辑器中自动补全 MCP 资源
  • 启动新会话时提供工作区控制
  • 会话上下文显示已存令牌和费用总计
  • 新增实时事件订阅流并提供 SDK 访问

问题修复

  • 强制推理模式对 OpenAI 兼容模型生效，确保推理设置可靠
  • 停止重放过期的 GitHub Copilot 响应 ID，避免请求失败
  • 让问题提示最小化后可恢复，不会被关闭
  • 清理空的旧草稿文件，减少桌面状态累积
  • 将会话页面错误限制在受影响的标签页，防止整个应用崩溃
  • OAuth 后重新连接 MCP 服务器，即使服务器被禁用
  • 显示 MCP OAuth 完成错误而非通用失败
  • 自动完成已配置的提示引用
  • 恢复编辑器重新获得焦点时的提示光标
  • 防止隐藏终端标签卡挂起或尺寸错误

Suwayomi/Suwayomi-Server v2.3.2223 GPT-OSS 120B

新内容

  • 新增 SyncYomi 支持，可与其他 Mihon 分支同步漫画
  • 支持 Extension API v1.6，兼容旧扩展并可迁移至 Extension Store
  • OPDS 新增跳过章节元数据的选项，直接提供流媒体 / 下载链接
  • Tracker API 新增绑定已有追踪记录的变更
  • 更新 WebView 使用 JCEF 并升级至最新 Chromium 引擎

问题修复

  • 修复 CloudFlareInterceptor 不再发送 cf_clearance cookie
  • 修复 WebUI 在非默认界面时的加载问题
  • 修复下载文件名因字节长度导致的 “文件名过长” 错误
  • 修复章节更新失败导致的章节数据丢失
  • 修复 GraphQL 过滤器 notAll/notAny 逻辑颠倒的问题

Breaking Changes

  • 扩展仓库将迁移至 Extension Store，更新前请备份仓库数据
  • 将 MessageQueue 从 ConcurrentMessageQueue 切换为 Legacy

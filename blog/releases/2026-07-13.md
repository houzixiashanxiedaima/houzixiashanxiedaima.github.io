### [MHSanaei/3x-ui v3.5.0](https://github.com/MHSanaei/3x-ui/releases/tag/v3.5.0) <sup>gpt-oss-120b</sup>

**新内容**
- MTProto 多客户端支持，采用 mtg-multi 引擎，实现 per-client secret、广告标签、配额与到期管理
- SQLite → PostgreSQL 完整迁移，面板可直接导入 SQLite .db 文件和迁移转储，新增 pgclient 命令
- 面向 50 万客户端的性能提升：批量 IP 限制查询、客户端 ID 禁用、增量 WebSocket 统计等
- 前端平台全面升级：React Hook Form、Fetch 替代 axios、图表迁至 uPlot、加入 Husky 等开发工具
- 节点同步加固：主节点接管 host 覆盖、避免同步死锁、编辑不再拆除节点、自动续期打开新配额窗口
- 出站信息增强：新增 IP+国家元数据、真实延迟连接测试、targetStrategy 字段
- 路由与负载均衡改进：支持负载均衡间回退、默认出站、加密 DNS 预设、私有 IP DNS 通过
- 面板体验优化：入口列表和节点选择文本搜索、列排序、实时速度显示、批量添加主机、WireGuard 导出分为配置与链接
- 更新 Xray-core 至 v26.7.11 并适配面板
- 数据库迁移修复流量计数溢出并移除 inbound 端口唯一约束
- 前端新增多项功能：批量添加主机、WireGuard 客户端 allowedIPs 可编辑、实时速度、客户端分组展示等
- 订阅页面新增公告显示并提供 HTML 信息页
- UI 新增清除已存储密钥的操作
- Telegram Bot 增加注册使用、重启、清除全部等命令并显示主机名
- 其他细节改进：密码字段协议说明、隐藏已禁用的 inbound、客户端信息模态框显示分组等

**Breaking Changes**
- MTProto 单密钥旧版自动迁移为多客户端模型，tg:// 深链不再携带 remark 片段
- Final Mask + REALITY 组合被拒绝，不再支持该组合

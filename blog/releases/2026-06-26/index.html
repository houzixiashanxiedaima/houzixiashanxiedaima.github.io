google-gemini/gemini-cli v0.49.0 GPT-OSS 120B

新内容

  • 支持 GDC 空气隔离服务身份
  • 添加静态评估源分析器
  • 新增 eval:inventory CLI 命令及报告功能
  • 标准化工具输出格式

问题修复

新内容

  • 通过 Crowdin 改进翻译

问题修复

  • 修复因搜索结果缓存未清除导致的高内存消耗
  • 修复版本升级时的部分安装程序失败情况

leoafarias/fvm 4.1.2 GPT-OSS 120B

  • 新增 OpenCode AI 翻译接口支持
  • 收藏单词页新增 AI 词典标签页，便于查看不同词典结果
  • 优化 YouTube 字幕 AI 断句与分块调度，减少提前处理
  • 优化字幕界面显示、悬停查找、列表标签和层级表现

问题修复

  • 修复已处理节点可能被重复翻译的问题，并优化内置 AI 超时处理

rorkai/App-Store-Connect-CLI 2.4.0 GPT-OSS 120B

新内容

  • 迁移维护者工作流至 agent skills
  • 添加结构化遥测错误上下文

jxxghp/MoviePilot v2.13.15 GPT-OSS 120B

新内容

  • 新增消息编辑功能并支持按钮描述
  • 新增文件处理时的传输历史检查

  • 更新 moviepilot-rust 依赖至 0.1.11

问题修复

  • 支持自定义剧集偏移表达式
  • 在解析前解码 XML feed
  • 尊重显式订阅最佳版本设置
  • 修复观众加入日期获取问题

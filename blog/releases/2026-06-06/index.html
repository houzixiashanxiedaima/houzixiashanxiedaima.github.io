KartikLabhshetwar/better-shot v0.3.3 - v0.3.6 GPT-OSS 120B

新内容

  • 新增设置页面 “恢复默认” 按钮，支持重置效果、通用设置和快捷键
  • 恢复捕获设置中的预览覆盖位置和自动消失延迟选项
  • 在捕获设置中新增 “恢复默认快捷键” 按钮
  • 快捷键录制器改进，临时禁用 CGEvent tap，避免触发系统截图
  • 检查器面板移至左侧，编辑区在右侧
  • README 更新，加入 Homebrew 安装指令并去除硬编码版本号
  • Landing 页面新增颜色选择、自拍计时、二维码 / 条码扫描等功能
  • CONTRIBUTING.md 更新，展示左侧检查器的编辑流程图
  • llms.txt 重写，准确描述 Swift 6/SwiftUI 架构及完整功能列表
  • 新增启动时自动检查更新并在菜单栏显示蓝点徽章
  • 捕获后智能提示：复制到剪贴板时显示 “已保存并复制”，否则显示 “已保存”
  • 简化捕获流程，默认设置直接保存，无需切换编辑 / 画廊模式
  • 截图后同时弹出浮动预览面板和提示通知
  • 在菜单栏点击最近截图可直接打开编辑器
  • 新增自定义弹出菜单栏，取代默认 MenuBarExtra，提供 toast 通知、设置覆盖层和捕获控
  • 新增窗口捕获功能，可通过菜单栏或快捷键捕获指定窗口
  • 新增签名发布构建，支持双架构的签名、公证并生成 DMG
  • 改进菜单栏图标渲染，优化模板显示并自动检测 URL
  • 改进编辑画布，提升缩放精度和标注框定位

问题修复

  • 隐藏设置页面的滚动条
  • 使用 ZStack 保持设置标签页状态切换时不丢失
  • 历史记录标签页异步加载缩略图，避免阻塞 UI
  • 最近捕获点击后显示浮动预览面板
  • 打开编辑器或设置时避免切换到错误的 Space
  • 编辑器检查器侧栏隐藏滚动条
  • 设置页面 About 选项卡隐藏滚动条
  • 修复窗口关闭时因 setActivationPolicy 导致的 EXC_BREAKPOINT 崩溃
  • 修复编辑器打开时出现双层背景的问题
  • 修复编辑器和设置窗口在错误的桌面打开的问题
  • 解决 7 处竞争条件，防止菜单栏弹出、快捷键等导致的崩溃
  • 修复屏幕捕获逻辑，清理捕获流程并重构菜单栏按钮

Breaking Changes

  • 移除未实现的 “重复区域” 快捷键 (Cmd+Ctrl+Shift+4)
  • 移除截图模式切换，统一使用自动应用默认设置的画廊模式

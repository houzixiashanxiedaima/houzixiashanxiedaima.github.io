SagerNet/sing-box v1.13.2 Qwen3 8B

新内容

  • 优化了整体性能和稳定性

问题修复

  • 修复了多个已知问题

jdx/mise v2026.3.4 Qwen3 8B

新内容

  • 新增运行时 musl/glibc 检测以正确选择 libc 变体
  • 新增交互式任务字段以提供专属终端访问
  • 生成的 lockfiles 现在包含 @generated 头部注释
  • 在 Windows 上保留 .exe 扩展名
  • JSON schema 支持任务依赖中的结构化对象

问题修复

  • 修复注册表平台选项在安装时未应用的问题
  • 修复工具选项以原生 TOML 格式存储以支持平台切换
  • 修复设置 MISE_INSTALL_PATH 为目录时的安装错误
  • 修复 prepare 源和输出路径相对路径解析问题
  • 修复旧版本 Ruby 预编译二进制查找问题

triwinds/ns-emu-tools 0.6.0 Qwen3 8B

新内容

  • 后端重构为 Rust 加 Tauri，显著减小 exe 体积
  • 新增内置 Rust 下载器，支持 Auto / Aria2 / Rust 模式

Breaking Changes

  • Citrion 项目已关闭

fastfetch-cli/fastfetch 2.60.0 Qwen3 8B

新内容

  • 新增 {cwd} 支持，用于自定义标题格式
  • 新增 Zed 版本检测功能
  • 新增 moss 包检测支持
  • 新增对 komorebi、FancyWM 和 GlazeWM 的支持
  • 新增 macOS 上 WM 插件版本检测支持

问题修复

  • 修复因负索引引用导致的潜在段错误
  • 修复 tempSensor 解析，仅接受字符串值
  • 修复 Linux 上键盘设备报告不全的问题
  • 优化 LXQt 的 WM 检测逻辑，仅在未检测到 WM 时查询设置
  • 修复 DBus 连接处理和 OpenGL EGL 上下文生命周期中的内存泄漏

microsoft/vscode-copilot-chat v0.38.2 Qwen3 8B

问题修复

  • 修复了 0.37 版本中的问题

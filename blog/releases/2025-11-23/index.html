sst/opencode v1.0.96 - v1.0.105

新内容

  • 支持 TUI 中的 Ctrl-Z 终端挂起：现在用户可以在终端用户界面 (TUI) 中使用 Ctrl-Z 挂起当前会话，方便进行多任务处理。【影响范围：所有 TUI 用户】
  • 增强消息时间戳：当消息不在当前日期时，时间戳会显示具体日期，便于用户追踪历史记录。
  • 增加代理循环结束时间显示：在代理循环中显示结束时间，帮助用户更好地管理任务。
  • 更新标题生成提示：优化了标题生成的提示词，防止生成 “总结” 类的响应，确保内容更加准确。
  • 影响范围*：所有使用 sst/opencode 的用户将受益于这些改进，特别是需要详细时间管理和可视化展示的场景。
  • 添加了缺失的系统库到 Docker 镜像中，确保代理能够成功启动。【影响范围：所有使用 Docker 部署的用户】
  • 此版本主要解决了 Docker 镜像中缺少必要系统库的问题，确保代理可以正常运行。建议所有使用 Docker 部署的用户尽快升级。

问题修复

  • 修正了会话视图中消息完成时间和持续时间的显示问题，确保时间信息准确无误。
  • 解决了用户界面在某些操作下出现的闪烁现象，提升了用户体验。
  • 修正了一个导致助手页脚在数据压缩后崩溃的错误，确保了应用的稳定性。
  • 此次更新解决了用户在使用 TUI（文本用户界面）进行工作会话时遇到的屏幕闪烁和布局不稳定的问题，提升了用户体验。

Breaking Changes

  • 性能优化对所有用户都有积极影响。

pppscn/SmsForwarder v3.5.0 Qwen 2.5 7B

新内容

  • 新增：远程唤醒 (WOL) 功能
  • 新增：呼叫转移标识
  • 新增：{{CONTACT_NAME}} 标签
  • 新增：{{PHONE_AREA}} 标签
  • 新增：闪烁手机功能
  • 新增：关屏功能

问题修复

  • 修复：PGP 加密邮件签名问题
  • 修复：免打扰脏数据问题
  • 修复：Bark 配置保存问题
  • 修复：未授权访问提示
  • 优化：Bark 消息链接变量
  • 优化：Telegram MarkdownV2 解析

Breaking Changes

  • v3.5.0 开始，应用包名改为 cn.ppps.forwarder，升级后需重新配置
  • v3.2.0 去除 mmkv 依赖，采用 SharedPreferences 保存配置，需重新配置

fastfetch-cli/fastfetch 2.59.0 GLM-4-Flash

新内容

  • 添加 Oracle Solaris 支持
  • 添加 UID / SID 检测
  • 自定义格式：{user-id}
  • 在 GNU/Hurd 上切换到原生 GPU 检测并移除 libpciaccess 依赖
  • 改进 macOS 内存大小检测
  • 避免默认依赖 hw.memsize_usable，可能不在旧版 macOS 上可用
  • 改进 Windows 磁盘检测准确性和性能
  • 添加更多 ARM CPU 部分，并移除重复案例
  • 为 NixOS Logo 添加 6 色支持（包括小尺寸版本）

louislam/uptime-kuma 2.1.1 GLM-4-Flash

新内容

  • 添加可自定义的矩阵通知
  • 在仪表板上添加监控器的组名列
  • 更新翻译

问题修复

  • 修复 Gamedig 无法解析 SRV 记录的问题
  • 移除 vite-plugin-pwa 插件，避免意外缓存文件
  • 修复 discord 通知时间戳和更新通知消息
  • 添加滚动条到标签下拉菜单
  • 仅当 TLD 支持 RDAP 时启用域名到期通知

ViennaRSS/vienna-rss v/3.10.0 GLM-Z1-Flash

新内容

  • 中键关闭标签页
  • 添加最近关闭标签页恢复功能
  • 新增通知设置访问按钮
  • macOS Tahoe 视觉适配
  • 使用绿色双点标记修订文章

问题修复

  • 解决部分 RSS 检测导致的程序冻结
  • 修正文章最后更新时间显示异常
  • 防止文件夹顺序重置为字母排序
  • 修复搜索插件兼容性问题
  • 改进 AppleScript 支持

anomalyco/opencode v1.1.65 GLM-4-Flash

新内容

  • 添加 tool.definition 钩子，允许插件修改工具描述和参数
  • 在设置中切换所有提供者模型
  • 清理桌面加载页面
  • 优化桌面断开重连事件流
  • 修复提示输入行为的小问题
  • 仅当需要 SQLite 迁移时显示加载窗口

问题修复

  • 撤销 OPENCODE_CONFIG_CONTENT 中的令牌替换
  • 确保使用读取工具而不是已弃用的列表工具来提及目录
  • 解决 ACP 在 Windows 上无限期挂起的问题
  • 修复桌面应用中创建存储失败的问题

RSSNext/Folo desktop/v1.3.0 GLM-4-Flash

anomalyco/opencode v1.14.42 - v1.14.46 DeepSeek V4 Flash

新内容

  • 新增内置 customize-opencode 技能，降低配置编辑导致启动失败的风险
  • TUI 插件兼容已弃用的 api.command API，同时支持迁移至 api.keymap
  • 新增 HTTP API 响应压缩支持
  • 新增 Scout 智能体用于仓库研究
  • 新增工作区同步自动发现适配器
  • 新增交互式分页底部模式
  • 简化 TUI 键位配置为扁平格式

问题修复

  • 修复 HTTP API 查询参数中数字类型在 OpenAPI 规范与 SDK 中的问题
  • 修复布尔型 HTTP API 查询参数处理，使 SDK 类型与运行时验证一致
  • 兼容旧版会话、差异和重试事件中的数字存储值，避免加载失败
  • 修复旧会话中负 token 计数导致消息加载和桌面端启动失败的问题
  • 修复 MCP 工具发现中服务器发布错误 outputSchema 引用的问题
  • 修复工作区 HTTP API 查询参数漂移，确保 OpenAPI 和 SDK 暴露正确参数
  • 修复 Plan Mode 安全绕过漏洞，防止子代理忽略父代理的拒绝规则
  • 提供者配置和 API 响应现在接受标记为活跃的模型
  • 读取工具权限规则现在匹配工作树相对路径
  • 工作区路由的 HTTP API 端点不再拒绝有效查询参数
  • 启动错误现在报告每个失败的引导请求
  • 打开会话时消息请求失败不再崩溃
  • 缺少差异文件详情的旧迁移会话可重新加载
  • 缺少差异补丁的旧迁移会话可重新加载
  • throwOnError 现在抛出包含服务器消息的真实错误
  • 提供者插件不再能变异共享的提供者模型状态
  • 修复会话中差异的可选补丁字段
  • 修复现有工作区添加 time_used 字段时升级失败的问题
  • 修复认证加载器为非 JSON 选项时 provider 和 config API 响应
  • 在 ACP 更新和会话回放中包含工具图片附件
  • 修复 HTTP API 认证响应格式问题
  • 返回 HTTP API 结构化验证错误
  • 拒绝 HTTP API 中无效权限和问题 ID
  • 修复 Gemini 推理控制与模型匹配
  • 修复 Anthropic Opus 4.5 推理选项

MHSanaei/3x-ui v3.0.0 DeepSeek V4 Flash

新内容

  • 前端全面重写为 Vue 3 + Ant Design Vue 4 + Vite 8
  • 新增多节点部署功能，支持远程节点管理
  • WebSocket 驱动实时更新替代旧轮询机制
  • 新增回环出站协议和增强 DNS 设置
  • 移动端适配：入站和客户端卡片布局、日志弹窗

问题修复

  • 修复 webBasePath 在开发和生产环境中的端到端问题
  • 修复离线时更新检查的警告日志刷屏问题
  • 修复面板更新后重启和深色模式版本标签问题
  • 修复 WebSocket 客户端陈旧事件和断开竞争问题
  • 修复移动端仪表盘布局问题

Breaking Changes

  • 翻译文件从 TOML 迁移至 JSON，需更新自定义翻译
  • 移除 Reality 中的 tesla.com 域名（因被屏蔽）
  • 跳过 Xray 26.5.3 版本，最低版本要求变更
